Research and Internship Projects

Vision Correcting Display

Advised by Prof. Brian Barsky, designed, prototyped, and characterized light field display algorithms to enable convolutional prefiltering of vision correcting displays that pre-distorts the presented content for visually impaired people through the integration of hardware and software.

Visual Layout Transfer

Advised by Prof. Bjoern Hartmann and graduate student Jeremy Warner, designed a graph-kernel-based method and developed an interactive UI for vector graphics layout transfer, essentially enabling content consistency while extracting relational rules from a source design and applying those layout rules to a target design.

3D Image Registration

Advised by Prof. Laura Waller and graduate students Leyla Kabuli, Ruiming Cao, and Tiffany Chien, implemented and validated a Multi-Layer Perceptron (MLP) based model for 3D medical image registration, with a focus on defining the neural network as a space-time MLP and a motion MLP, followed by memory optimization.

Light Field 3D Recontruction

Worked on light field 3D reconstruction algorithm based on mobile captured images for Kuaishou Flagship, a short video social networking platform with 100M+ users, boosting modeling efficiency from 1 week to 4-6 hours. Created face tracking 3D filters and developed the UI to streamline object capturing and model generation workflow.

Computer Graphics and Computer Vision Projects

Color Image Production

Implemented RGB color images from digitized Prokudin-Gorskii glass plate images using fast search procedure image pyramid and similarity metrics Sum of Squared Differences (SSD) and Normalized Cross-correlation (NCC). With these image processing techniques, I extracted the three color channel images, placed them on top of each other, and aligned them so that they form a single RGB color image.

Rasterizer

Implemented an image rasterizer from rasterizing triangles (the basic of rendering), supersampling, transforming, using Barycentric coordinates to linear interpolation, bilinear sampling and trilinear sampling.
